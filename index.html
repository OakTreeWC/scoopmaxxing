Scoopmaxxing 101

Do YOU. Yes, YOU, want to learn how to start Scoopmaxxing? Well then, listen up folks, cause this website, will teach you everything you need to know to start doing just that.
According to the requirements for scoops, examples to start earning scoops include but are not limited to,

  1. having an HTML footer
  2. having a link to another website
  3. having a special link (email, phone, sms, etc)
  4. having special text formatting (bold, italics, underlines, highlighting, etc)
  5. having an HTML list
  6. Having bookmarks (ignore this one)
  7. Having a 404 page
  8. Having a custom font
  9. Having a HTML button
  10. Having a CSS :hover or other pseudo-class
  11. Having HTML Audio.
